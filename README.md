# GRAMODAY 

## Instructions

1. Install the packages in the requirements.txt file and run the Task1_Scraping.py file
   ```
   pip install -r requirements.txt
   python Task1_Scraping.py
   ```

2. The script is running successfully on Linux. Scraped Data will be saved in '2020_Potato_Agra_Price.csv' file.

Note : If you're on Windows/Mac, please download the ChromeDriver version according to your system's specifications.
